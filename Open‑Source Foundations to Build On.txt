Install Eramba CE Using Docker (Recommended)
This is the easiest, cleanest, and officially supported method.
The Eramba site explicitly points users to their Docker Install Guide. 

Prerequisites
Docker Engine

Docker Compose

A Linux VM or your local machine (Ubuntu recommended)

Step-by-Step Installation
1. Create a directory for Eramba
bash
mkdir eramba
cd eramba
2. Create a docker-compose.yml file
Use this template (from the official Docker install guidance):

yaml
version: '3.3'
services:
  eramba:
    image: eramba/community:latest
    container_name: eramba
    ports:
      - "80:80"
    volumes:
      - eramba_data:/var/www/eramba
    restart: always

volumes:
  eramba_data:
3. Start Eramba
bash
docker compose up -d
